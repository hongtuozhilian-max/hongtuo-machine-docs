This document describes the basic operating procedure
for the HL-800F remote-controlled flail mower.

## Pre-Operation Check
Before starting the machine, the operator should ensure:

- The machine is placed on stable ground
- Fuel level is sufficient
- Engine oil level is within the normal range
- Hydraulic system shows no visible leakage
- Tracks are free from debris
- The cutting area is free of large obstacles

## Starting Procedure
1. Place the operator at a safe distance from the machine
2. Switch on the remote controller
3. Start the engine via remote control
4. Allow the engine to warm up for several minutes
5. Engage the cutting system only after stable engine operation

## Basic Operation
During operation, the operator can control:

- Forward and backward movement
- Left and right turning
- Cutting system on / off
- Cutting height adjustment

It is recommended to:
- Operate at low speed on slopes
- Maintain a stable and smooth movement pattern
- Avoid sudden direction changes on steep terrain

## Slope Operation Guidelines
When operating on slopes:

- Always keep the machine oriented uphill or downhill
- Avoid lateral movement on steep slopes
- Maintain a safe distance between operator and machine
- Use gradual speed adjustments

## Stopping Procedure
1. Disengage the cutting system
2. Bring the machine to a complete stop
3. Stop the engine via remote control
4. Switch off the remote controller
5. Allow the machine to cool down before inspection

## Emergency Stop
The HL-800F is equipped with an emergency stop function.

In case of emergency:
- Activate the emergency stop immediately
- The engine and cutting system will shut down instantly
- Do not restart the machine until the issue is identified
